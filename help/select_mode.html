Select running mode

This page allows you to select whether you are running the MWC Qt Wallet as a standard online hot wallet (the default choice and what most users will use), An "Online Node" which is used to submit transactions from cold storage, or a "Cold Wallet" which is used in conjunction with the "Online Node" and installed on a computer that never connects to the internet. Transactions are sent/received/finalized on the "Cold Wallet" but submitted with the "Online Node"

More details about Cold Wallet usecase you can find at our Cold Wallet Guide
https://github.com/mwcproject/mwc-qt-wallet/blob/2019_09_cold_wallet/DOC/cold_wallet.md
