Transactions

The transaction page shows all transactions that have been made from this wallet.

Please note, if you use the recover function to recover from a mnemonic, you will not be able to see your transaction history.

Every transaction has:

  • Type of the transaction (Receive, Send or Cancelled) and it status (Confirmed, Unconfirmed, Cancelled).
  • Amount in MWC.
  • UUID identifier of the transaction.
  • The address of the other party to the transaction if it is known. Files and unencrypted Slatepacks transactions can be anonymous.
  • Time when the transaction was made or confirmed.
  • Number of confirmations for confirmed transactions.
  • Confrimation - whether or not the transaction has been confirmed to the blockchain.
  • Optional transaction note.
