Outputs Tab

The outputs tab lists the outputs that are currently available to spend in this wallet instance.

Fields are as follows:

  • Status of the output: (Unspent, Spent, Locked, Unconfirmed)
  • Block height that output belongs to.
  • The amount of MWC in this output.
  • The commitment associated with this output.
  • Number of confirmations that commitment already get.
  • The optional Lock height. The height after which this output may be spent.
  • Optional output note.
