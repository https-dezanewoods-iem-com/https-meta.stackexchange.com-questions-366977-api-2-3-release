The transaction page shows all transactions that have been made from this wallet.

Please note, if you use the recover function to recover from a mnemonic, you will not be able to see your transaction history.

Every transaction has:

  • Type - Receive, Send or Cancelled.
  • ID - An identifier of the transaction.
  • Address - The address of the other party to the transaction.
  • Time - Time when the transaction was made.
  • Amount - Amount in MWC.
  • Confrimation - whether or not the transaction has been confirmed to the blockchain.
